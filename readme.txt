== Description ==
Preloader is used when pages are loding whie processing it displays a loader.

== How to Use ==

After installation it will automatically displays a loader on the site when you click to open any page.

== Requirements ==
It can run on any wordpress version.

== Frequently Asked Questions ==

= How can I change background image for loader? =
  You can pass your image url to load any image.

= What happend if i will not upload image ?
  It will work with default loader image.
